Brand compare

This view helps a chain's strategy team answer: “Where are we strong/weak vs competitors, and how concentrated is each brand?”
Note: prototype uses outlet counts only. In production, we'd add delivery coverage, openings/closures, ratings, price index, and platform visibility.

How to use

  1. Pick brands to include (left panel in Overview).
  2. Use the map to identify high-intensity regions.
  3. Click a region to see brand mix + top cities.
  4. Export filtered data as CSV (Export tab) for further analysis.
Example use case
A pizza chain can compare Domino's vs Papa Johns region-by-region, find whitespace cities, then prioritize new store pipeline.
